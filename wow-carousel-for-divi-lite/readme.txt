== Description ==

**Divi Carousel Free** is a carousel plugin designed exclusively for the Divi Theme and Divi Builder. Create stunning, responsive carousels in minutes without writing a single line of code.

Whether you need an image gallery carousel or a logo showcase, this plugin integrates seamlessly with Divi's visual builder interface.

= Why Choose Divi Carousel Free? =

* **Easy to Use** – Built for Divi with a familiar drag-and-drop interface
* **Lightning Fast** – Optimized for performance with minimal resource loading
* **Fully Responsive** – Adapts to desktop, tablet, and mobile screen sizes
* **Touch-Enabled** – Swipe gestures for smooth mobile navigation
* **Customizable** – Full control over layout, design, and behavior
* **Divi 5 Ready** – Compatible with both Divi 4 and Divi 5

= Included Modules =

**Image Carousel**
Create image carousels with customizable transitions, autoplay, navigation arrows, and pagination dots. Ideal for photo galleries, product showcases, and portfolio presentations.

**Logo Carousel**
Display client logos, partner brands, or sponsor images in a professional carousel layout. Includes custom links, hover effects, and flexible spacing controls.

== Installation ==

= Automatic Installation =

1. Log in to your WordPress dashboard.
2. Navigate to Plugins > Add New.
3. Search for "Divi Carousel Free".
4. Click "Install Now" and then "Activate".
5. Open the Divi Builder and search for "Carousel" to start using the modules.

= Manual Installation =

1. Download the plugin zip file.
2. Go to Plugins > Add New > Upload Plugin.
3. Choose the downloaded zip file.
4. Click "Install Now" and then "Activate Plugin".
5. Access the carousel modules through Divi Builder.

== Frequently Asked Questions ==

= Does Divi Carousel work with any WordPress theme? =

No. Divi Carousel requires the Divi Theme or Divi Builder plugin by Elegant Themes to function.

= Is Divi Carousel responsive? =

Yes. All carousels are fully responsive and include touch-swipe gestures for mobile devices.

= Will Divi Carousel slow down my website? =

No. The plugin only loads resources on pages where carousels are used, keeping your site fast.

= What is the difference between the free and pro versions? =

The free version includes the Image Carousel and Logo Carousel modules with core carousel features. Divi Carousel Pro adds additional modules (Content, Post, Testimonial, Team, WooCommerce) and advanced features like 3D effects, custom pagination styles, and priority support.

= Does Divi Carousel support RTL languages? =

Yes. Full RTL (Right-to-Left) support is included for languages like Arabic and Hebrew.

= Where can I get support? =

Free support is available through the [WordPress.org support forum](https://wordpress.org/support/plugin/wow-carousel-for-divi-lite/). Pro users receive priority email support.
